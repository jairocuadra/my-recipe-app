Underwrite Pro UI Component Library Documentation

Introduction

The Underwrite Pro UI Component Library is a collection of reusable Angular components designed to provide a consistent look and feel across the Underwrite Pro application. This documentation provides information on how to use these components in your application.

Installation

To use the Underwrite Pro UI Component Library in your Angular project:


// Import the module in your app.module.ts
import {{ '{' }} UnderwriteProUiModule {{ '}' }} from 'underwrite-pro-ui';

@NgModule({{ '{' }}
  imports: [
    // other imports
    UnderwriteProUiModule
  ],
  // ...
{{ '}' }})
export class AppModule {{ '{' }} {{ '}' }}

Components

Card Component

A versatile card component for displaying content in a contained format.

Usage


<uwp-card 
  title="Card Title" 
  [subtitle]="Optional subtitle"
  [elevation]="'low' | 'medium' | 'high'"
  [padding]="'small' | 'medium' | 'large'"
  [noBorder]="true | false">
  
  <!-- Card content goes here -->
  <p>This is the card content.</p>
  
</uwp-card>

Properties

Name Type Default Description
title string undefined The title of the card.
subtitle string undefined The subtitle of the card.
elevation 'low' | 'medium' | 'high' 'medium' The elevation (shadow) of the card.
padding 'small' | 'medium' | 'large' 'medium' The padding inside the card.
noBorder boolean false Whether to remove the border from the card.

Section Title Component

A component for creating consistent section titles throughout the application.

Usage


<uwp-section-title 
  title="Section Title" 
  [subtitle]="Optional subtitle"
  [color]="'primary' | 'accent' | 'default'"
  [withDivider]="true | false">
</uwp-section-title>

Properties

Name Type Default Description
title string required The title text.
subtitle string undefined The subtitle text.
color 'primary' | 'accent' | 'default' 'primary' The color of the title.
withDivider boolean true Whether to show a divider below the title.

Button Component

A customizable button component with various styles and states.

Usage


<uwp-button 
  label="Button Label" 
  [type]="'button' | 'submit' | 'reset'"
  [variant]="'primary' | 'secondary' | 'outline' | 'text'"
  [size]="'small' | 'medium' | 'large'"
  [icon]="'icon-name'"
  [iconPosition]="'left' | 'right'"
  [fullWidth]="true | false"
  [disabled]="true | false"
  [loading]="true | false"
  (buttonClick)="handleClick($event)">
  
  <!-- Optional icon content -->
  <mat-icon icon-left>home</mat-icon>
  
</uwp-button>

Properties

Name Type Default Description
label string required The button label text.
type 'button' | 'submit' | 'reset' 'button' The HTML button type.
variant 'primary' | 'secondary' | 'outline' | 'text' 'primary' The button style variant.
size 'small' | 'medium' | 'large' 'medium' The button size.
icon string undefined The name of the icon to display.
iconPosition 'left' | 'right' 'left' The position of the icon relative to the label.
fullWidth boolean false Whether the button should take up the full width of its container.
disabled boolean false Whether the button is disabled.
loading boolean false Whether to show a loading spinner in the button.

Events

Name Type Description
buttonClick EventEmitter<MouseEvent> Emitted when the button is clicked.

Form Field Component

A form field wrapper component for consistent form styling.

Usage


<uwp-form-field 
  label="Field Label" 
  [labelFor]="'input-id'"
  [helperText]="'Helper text goes here'"
  [errorText]="formControl.errors ? 'Error message' : null"
  [required]="true | false"
  [fullWidth]="true | false"
  [disabled]="true | false">
  
  <!-- Input element goes here -->
  <input id="input-id" type="text">
  
</uwp-form-field>

Properties

Name Type Default Description
label string undefined The label text for the form field.
labelFor string undefined The ID of the input element this label is for.
helperText string undefined Helper text to display below the input.
errorText string undefined Error message to display when the input is invalid.
required boolean false Whether the field is required.
fullWidth boolean false Whether the field should take up the full width of its container.
disabled boolean false Whether the field is disabled.

Theming

The Underwrite Pro UI Component Library supports both light and dark themes. To use the dark theme, add the dark-theme class to a parent element: 


<div class="dark-theme">
  <!-- Components will use dark theme styles -->
  <uwp-card title="Dark Theme Card">
    <p>This card will use dark theme styles.</p>
  </uwp-card>
</div>
